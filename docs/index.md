## Overview
LUCKY3D is a modern, tabbed 3D viewer with a clean UI, drag-and-drop loading, and an informative panel to inspect files and scene properties. It features a responsive toolbar, a tabs system for managing multiple files, and a drop zone for quick imports.

If you are viewing this site on GitHub Pages, it is generated from the `docs/` folder using the Cayman theme.

## Project Structure
The project contains:
- `index.html`: Main application page
- `styles.css`: Global styles for the app UI
- `main.js`, `renderer.js`: Application logic
- `assets/`: Images and other static resources
