# Shortcuts

Below are suggested/common shortcuts. Exact keys depend on your implementation in `main.js`/`renderer.js`.

- Open file: `Ctrl+O` (or drag-and-drop).
- Close tab: `Ctrl+W` (active tab).
- Next tab: `Ctrl+Tab`.
- Previous tab: `Ctrl+Shift+Tab`.
- Toggle info panel: `I`.
- Reset view: `R`.
- Help / shortcuts: `?`.

> Adjust to match your actual bindings. If a shortcut isn't implemented yet, consider adding an on-key handler and updating this page.
