# Features

LUCKY3D focuses on usability and clarity for inspecting and working with 3D scenes/files.

## Core
- Tabbed interface for multiple files.
- Drag-and-drop import with centered drop zone and guidance.
- Information panel with scene/file metadata.
- Responsive toolbar with grouped actions and separators.
- Full-viewport canvas rendering with a dark theme.

## UI/UX
- Glow-accented buttons with hover/active transitions.
- Distinct tabs with close buttons and active highlighting.
- High-contrast info panel for quick inspection.
- Keyboard-friendly interactions (see Shortcuts).

## Architecture
- `index.html` hosts the main application shell.
- `main.js` / `renderer.js` handle core interactions and rendering.
- `styles.css` defines the app look and feel.

## Roadmap Ideas
- Additional import formats & converters.
- Scene graph inspector and property editor.
- Per-tab settings and persistent layout.
